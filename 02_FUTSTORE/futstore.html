Projeto comercial

Futstore

Site de venda de camisas de futebol desenvolvido com vitrine de produtos, visual moderno, modal de pedido e envio automático das informações do cliente por e-mail.

Ver demonstração Funcionalidades

Sobre o projeto

A Futstore foi criada como uma loja virtual simples e objetiva para apresentar camisas de futebol, facilitar o pedido do cliente e tornar o processo de compra mais rápido, bonito e organizado.

Tecnologias utilizadas

HTML CSS JavaScript Fire Base EmailJS

Funcionalidades

Vitrine de Produtos

Exibição das camisas disponíveis com imagem, nome, preço e botão de compra.

Modal de Pedido

Formulário para o cliente informar nome, e-mail, telefone, endereço e dados do produto escolhido.

Envio por E-mail

Integração com EmailJS para enviar automaticamente os dados do pedido para o e-mail configurado.

Demonstração

Na demonstração, apresento a navegação pela Futstore, mostrando a vitrine de produtos, a abertura do pedido e o envio das informações para o e-mail.

Resultado

  • Criação de uma loja virtual visual e responsiva.
  • Organização dos produtos em uma vitrine simples.
  • Formulário de pedido com dados do cliente.
  • Envio automático das informações por e-mail.
  • Projeto publicado e pronto para portfólio.

Meu papel no projeto

Desenvolvi a Futstore desde a estrutura visual até as funcionalidades, criando o layout, a vitrine de produtos, o modal de compra, a lógica em JavaScript e a integração com envio de pedidos por e-mail.
